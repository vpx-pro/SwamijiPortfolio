## ğŸš€ Deployment on Vercel

This website is optimized for deployment on Vercel. Follow these steps:

### Prerequisites
- Node.js installed
- Vercel CLI installed (`npm i -g vercel`)

### Deployment Steps

1. **Clone or download this repository**
   ```bash
   git clone <repository-url>
   cd pandit-satyendra-deo-sharma-portfolio
   ```

2. **Install dependencies** (optional, for development)
   ```bash
   npm install
   ```

3. **Deploy to Vercel**
   ```bash
   vercel --prod
   ```

4. **Alternative: Deploy via Vercel Dashboard**
   - Go to [vercel.com](https://vercel.com)
   - Import your repository
   - Deploy with default settings

### Manual Deployment
If you prefer manual deployment:
1. Zip all files (index.html, styles.css, script.js, vercel.json, package.json, images folder)
2. Upload to Vercel dashboard
3. Deploy
